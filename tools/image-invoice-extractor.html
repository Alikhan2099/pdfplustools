No. This is a browser-based . A production-grade version should add stronger OCR/AI, .xlsx export, line-item extraction, multi-language invoices, and better layout detection.

Does it export Excel?

It exports CSV, which opens directly in Excel, Google Sheets, and Numbers. A future version can export native .xlsx files.
