File Handling & Processing

Last updated: May 2026

Browser-based tools

Many PDFplusTools tools are designed to run directly in your browser. This means the file is selected on your device and processed locally by browser JavaScript.

Examples

Browser-based tools may include image-to-PDF, PDF merge, PDF split, image compression, image resizing, QR code generation and certain invoice tools.

Files are not intentionally stored

For browser-based tools, we do not intentionally upload or store your selected files on our server.

Third-party client libraries

Some tools use third-party client-side libraries for PDF, image, QR code or OCR functionality.

Sensitive files

Avoid using online tools for highly sensitive files unless you understand the tool and accept the risk.
