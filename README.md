# PDFplusTools

Final static website package.

How to test:
1. Open index.html in a browser.
2. Test the top navigation on desktop and mobile.
3. Test tools/image-to-pdf.html, tools/merge-pdf.html and tools/image-background-remover.html.
4. Upload the files to GitHub and deploy with Cloudflare Pages.

The website is structured as a free online PDF, image, invoice and utility tools site.
